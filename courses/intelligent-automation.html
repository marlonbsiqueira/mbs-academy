Program overview

What this program covers.

Most automation programs teach you how to use a tool. This one teaches you how to think about automation — which processes to target, how to design solutions that scale, how to govern what you deploy, and how to build the business case that gets it funded.

Every module draws on real delivery experience: the 200,000-communication cross-border programme at Stellantis, the R2R automation that compressed month-end close from 8 to 6 days, the Power Platform workflows that eliminated 3 days of weekly manual reporting. The tools are real. The outcomes are documented.

Program modules

01 The Automation Opportunity — How to Think Before You Build ~90 min
Why most automation programmes fail (and what they have in common)
The automation readiness matrix: rules-based, judgement-based, and AI-assisted processes
Process selection criteria: volume, stability, frequency, exception rate
The automation opportunity map — a framework for prioritising your pipeline
02 Building the Automation Business Case ~80 min
Calculating the real cost of manual processes (beyond FTE time)
ROI modelling for automation: savings, risk reduction, speed gains
Presenting automation proposals to finance and senior leadership
Business case template (real Stellantis format included)
03 RPA Fundamentals — Concepts, Tools and Selection ~95 min
How RPA works: bots, orchestrators, attended vs unattended
Tool landscape: Blue Prism, Automation Anywhere, UiPath — how to choose
RPA use cases in finance: AP processing, reconciliation, month-end reporting
What makes an RPA project succeed vs fail — lessons from real deployments
04 Power Automate for Finance Workflows ~110 min
Power Automate fundamentals: flows, triggers, conditions, actions
Building approval workflows for purchase orders, accruals, and journal entries
Automated financial reporting distribution (the Stellantis case: 70+ leaders, zero manual intervention)
Connecting to SharePoint, Teams, Outlook, and SAP data
Error handling, logging, and compliance documentation
05 VBA Mastery for Finance Operations ~120 min
VBA foundations: modules, subroutines, variables, loops, conditionals
Automating Excel: data manipulation, formatting, report generation
Working with multiple workbooks and external data sources
Building automated management reports and dashboards in Excel
Real VBA use cases from Stellantis finance operations
06 Python for Finance Automation ~105 min
Python foundations for non-developers: syntax, data types, basic logic
Pandas for financial data: reading, cleaning, transforming, and exporting
Automating Excel with openpyxl: generating reports from raw data
Connecting Python to SAP data exports
When to use Python vs VBA vs Power Automate
07 AI-Augmented Automation — Microsoft Copilot and Beyond ~85 min
What AI actually changes in automation (and what it does not)
Microsoft Copilot across the finance stack: Excel, Teams, Outlook, Power BI
AI document processing: invoices, contracts, approval documentation
Practical applications deployed at Stellantis: documentation automation, analytics augmentation
08 Designing for Scale — Architecture, Governance and Maintenance ~90 min
Building an automation portfolio that doesn't become technical debt
Documentation standards for audit-ready automation (SOX, GDPR requirements)
Change management and exception handling in production environments
The governance framework I use at Stellantis to manage 20+ live solutions
09 Building Your Automation Roadmap ~75 min
Sequencing your automation pipeline: quick wins first, strategic builds second
Stakeholder alignment: IT, Finance, Compliance, and Operations
Measuring automation impact: KPIs beyond FTE savings
Automation roadmap template (used in Stellantis programme planning)
Built from the Stellantis automation programme
